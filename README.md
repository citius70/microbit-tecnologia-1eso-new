# 🚀 Tutoriales Educativos para Micro:bit

Bienvenido al repositorio de **Tutoriales Micro:bit**, un espacio diseñado por y para docentes apasionados por la educación **STEM**. El objetivo de este proyecto es facilitar la enseñanza de la programación y la electrónica básica mediante guías estructuradas, claras y listas para usar en el aula.

## 🎯 Propósito del Proyecto
Este repositorio busca cerrar la brecha entre la tecnología y el aprendizaje, proporcionando recursos en formato **Markdown (.md)** que los profesores pueden integrar fácilmente en plataformas como GitHub, Classroom o sus propios blogs educativos.

## 📚 Estructura de las Lecciones
Cada tutorial sigue un estándar pedagógico para asegurar el éxito del aprendizaje:
* **Introducción:** Contexto y qué se construirá.
* **Requisitos:** Hardware necesario (placas, sensores, cables).
* **Conceptos Clave:** Explicación teórica de los fundamentos técnicos.
* **Paso a Paso:** Instrucciones detalladas con bloques de código.
* **Desafío:** Ejercicios de extensión para fomentar el pensamiento crítico.

## 🛠️ Tecnologías Cubiertas
* **Programación:** MakeCode (Bloques/JavaScript) y MicroPython.
* **Hardware:** Micro:bit (V1/V2), sensores externos, actuadores y comunicación por radio.

## Usar como extensión

Este repositorio puede ser añadido como una **extensión** en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **New Project**
* haga clic en **Extensiones** en el menú del engranaje
* buscar **https://github.com/citius70/microbit-tecnologia-1eso-new** e importar

## Edita este proyecto

Para editar este repositorio en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **Import** luego haga clic en **Import URL**
* pegue **https://github.com/citius70/microbit-tecnologia-1eso-new** y haga clic en importar
